# Mirror

A modern desktop application for monitoring and observing browser automation sessions in real-time using Selenium WebDriver BiDi protocol.

## Features

- **Multi-Browser Support** - Chrome, Firefox, and Edge with headless mode
- **Real-Time Monitoring** - Live viewport screenshots with smart change detection
- **Network Inspector** - Monitor HTTP/HTTPS requests, responses, headers, and timing
- **Browser Controls** - Headless mode, beta channels, isolated instances, and multiple tabs
- **Cross-Platform** - Native experience on Windows, macOS, and Linux
